# Legal-Tech-Logic-Distiller (LTLD) ⚖️🤖

> **"Methods, not Souls."** —— 本项目是一个实验性的法律科技方法论蒸馏工具。

## 📖 项目背景
本项目受 `colleague-skill` 启发，但基于法学伦理进行了重构。在《黑镜》式的技术恐惧下，我们认为 AI 不应成为人类灵魂的囚笼，而应成为专业知识的杠杆。

本项目将哈尔滨工业大学（HIT）法律科技实习生的**工作范式**、**Prompt 工程经验**与**法律逻辑框架**进行了脱敏封装，旨在探索如何在保护个人人格权的前提下，实现专业能力的数字化传承。

## 🛡️ 伦理红线 (Ethical Guardrails)
1. **去人格化**：拒绝提取任何涉及个人私生活、情感、语调等私密数据。
2. **算法透明**：所有生成的逻辑链路均可追溯，拒绝“黑箱操作”。
3. **反剥削约束**：内置 `philosophy.py` 防御模块，严禁将本 Skill 用于任何违反人类尊严的指令。

## 🚀 核心功能
- **法律大模型评测逻辑**：内置 LLM-as-a-Judge 的双轨迭代思维。
- **知识产权分析范式**：针对商标近似度、AI 训练数据合规性的判定逻辑。
- **伦理防御网**：自动识别并拦截具有“物化人类”倾向的剥削性指令。

## 🛠️ 快速启动
1. 安装依赖：`pip install -r requirements.txt`
2. 运行项目：`python main.py`
